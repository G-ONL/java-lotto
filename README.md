# java-lotto
로또 미션 진행을 위한 저장소

기능 명세

- 구입금액 입력 받기
- 구매한 로또 개수 출력
- 구매한 개수 만큼 로또 배열 출력
- 당첨 번호 입력 받기
- 보너스 볼 입력 받기
- 구매한 로또와 비교
- 당첨 통계 출력
- 상금 계산 ( + 수익률)

예외 상황

- 1000원당 1게임인데 그 이하의 단위가 들어올 시 (1100원) // 1개만 구입할 지, 다시 입력 받을 지
- 당첨 번호 중복입력 // 다시 입력 받기
- 보너스 볼( 당첨 번호 중복) // 다시 입력 받기
